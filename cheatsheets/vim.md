# Files

```vim
" Find and open a file in a new buffer
:find /usr/share/vim/**/*imrc 

" Open the file under the cursor
gf
```

# Windows

```vim
" Create a split
CTRL-W s
" Create a vertical split
CTRL-W v
" Move between windows
CTRL-W hjkl
" Close a window
CTRL-W q
" Move windows around
CTRL-W HJKL
" Resize a window
:resize 20<CR>
:vertical resize 50<CR>
```
